赤石の民衆 通常コスチューム素材リポジトリ

このリポジトリは 赤石の民衆 (sokomin.github.io) で公開している RED STONE の 通常コスチューム の画像・GIF アセットを保管しています。 GitHub Pages 経由で https://sokomin.github.io/sokomin_repository_costume2024/... として配信されます。

収録範囲

  • 通常コスチューム: コラボ、季節限定、コンテスト、初期コスチューム等
  • 各コスチュームは 最大 11 色 のバリエーションを収録

強化前/強化後の 2 形態を持つ秘奥コスチュームは別リポ sokomin_repository_costume に保管しています。

ファイル命名規約

costume/
├── <name>_s1.png             # デフォルト色 静止画
├── <name>_s1_idle.gif         # デフォルト色 立ちアニメ
├── <name>_s1_pal01.png        # 色違い (pal01〜pal10)
├── <name>_s1_pal01_idle.gif
└── ...
  • _pal01〜_pal10 = 色違いバリエ (デフォルト色はサフィックス無し)
  • .png = 静止画 / _idle.gif = 立ちポーズアニメ
  • 色違いを持たないコスチュームはデフォルト色のみ

使用方法

サイト側の HTML から URL でアセットを参照してください:

<img src="https://sokomin.github.io/sokomin_repository_costume2024/costume/Knight_cos1_s1_idle.gif">
